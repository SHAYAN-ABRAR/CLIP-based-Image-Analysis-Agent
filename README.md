# CLIP-based Image Analysis Agent

## Overview
The **CLIP-based Image Analysis Agent** is an AI-powered tool that leverages CLIP (Contrastive Language-Image Pre-Training) and Ollama for image recognition, description generation, and image search. It integrates multiple AI models to perform image classification, generate detailed descriptions, and find similar images through both text and image queries.

## Features
- **Image Recognition**: Classifies images into predefined categories using the CLIP model.
- **Image Description**: Generates detailed descriptions based on image content.
- **Image Search by Text**: Search for images based on textual descriptions.
- **Image Search by Image**: Find similar images based on an input image.
